Install the package from pip with:

```bash
pip install cesnet-tszoo
```

or for editable install with:

```bash
pip install -e git+https://github.com/CESNET/cesnet-tszoo
```

## Requirements
The `cesnet-tszoo` package requires Python >=3.10.
